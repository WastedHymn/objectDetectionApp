# objectDetectionApp
## Projenin Amacı 
Mobilden kamerayla çekilen veya galeriden seçilen fotoğrafın içindeki nesneleri işaretleyip, hangi nesne olduğunu belirten ve işlenmiş fotoğrafı tekrar istemciye yollayan bir sistem yapmak.
